# Sky Forms

> A **Single Page Application (SPA)** for creating and distributing forms.

> **⚠️ Work in Progress**
> Sky Forms is currently under active development. Features and APIs are subject to change.

## About

Sky Forms is being built as a **Single Page Application (SPA)**, where the application loads once and dynamically updates the interface without full page reloads. The focus is on delivering a fast, seamless experience for form creation, management, and distribution.

## Tech Stack

* HTML
* CSS
* Vanilla JavaScript
